## 1. ОБЩАЯ ИНФОРМАЦИЯ

### 1.1 Описание проекта
Language Assist - это MVP приложение для изучения языков с следующими функциями:
- Анализ текстов на китайском языке (токенизация, лемматизация)
- Определение уровня HSK слов
- Генерация карточек из текста
- Система повторения SM-2 (Spaced Repetition)
- Словарь пользователя
- Упрощение текстов до целевого уровня HSK

### 1.2 Архитектура
- **FastAPI Backend** (порт 8000) - REST API
- **Go Analyzer** (порт 8080) - микросервис анализа текста
- **SQLite/PostgreSQL** - база данных

## 3. НАЙДЕННЫЕ ОШИБКИ И НЕДОЧЕТЫ

### 3.1 КРИТИЧЕСКИЕ ОШИБКИ

#### 🔴 Ошибка #1: Несовместимость bcrypt и passlib
**Файл:** `requirements.txt`  
**Проблема:** Версия `bcrypt==4.1.2` (или 5.0.0) несовместима с `passlib[bcrypt]==1.7.4`

**Симптомы:**
```
ValueError: password cannot be longer than 72 bytes, truncate manually if necessary
AttributeError: module 'bcrypt' has no attribute '__about__'
```

**Влияние:** Невозможность регистрации пользователей, полная неработоспособность аутентификации

**Решение:** Изменить версию bcrypt на 4.0.1
```diff
- bcrypt==4.1.2
+ bcrypt==4.0.1
```

**Статус:** ✅ ИСПРАВЛЕНО

---

### 3.2 ОШИБКИ СРЕДНЕЙ ТЯЖЕСТИ

#### 🟡 Ошибка #2: Отсутствие обработки ошибок Go сервиса
**Файлы:** `services/go_client.py`, `routes/texts.py`

**Проблема:** При недоступности Go микросервиса API возвращает 503 ошибку без детальной информации

**Код:**
```python
if not go_result or go_result.get("status") != "ok":
    raise HTTPException(status_code=503, detail="Go analyzer service unavailable")
```

**Влияние:** Пользователь не понимает причину ошибки, нет fallback механизма

**Рекомендация:**
- Добавить логирование ошибок Go сервиса
- Реализовать fallback на локальную токенизацию (jieba)
- Добавить retry механизм

---

#### 🟡 Ошибка #3: Хардкод user_id = 1
**Файлы:** `routes/flashcards.py`, `routes/texts.py`, `routes/users.py`

**Проблема:** Во всех роутах используется `user_id: Optional[int] = 1`

**Код:**
```python
def get_flashcards(
    db: Session = Depends(get_db),
    user_id: Optional[int] = 1,  # Для MVP, потом из токена
):
```

**Влияние:** 
- Все пользователи видят данные друг друга
- Нет реальной аутентификации
- Уязвимость безопасности

**Рекомендация:** Реализовать JWT аутентификацию

---

#### 🟡 Ошибка #4: Примитивная токенизация китайского
**Файл:** `analyzer-go/main.go`

**Проблема:** Токенизация посимвольная, а не пословная

**Код:**
```go
func tokenizeChinese(text string) []string {
    var tokens []string
    for _, r := range text {
        tokens = append(tokens, string(r))
    }
    return tokens
}
```

**Влияние:** 
- Слова разбиваются на отдельные иероглифы
- "学习" (учиться) становится "学" + "习"
- Неправильный анализ HSK уровня

**Рекомендация:** Использовать готовую библиотеку (например, go-jieba)
